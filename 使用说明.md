# AI Canvas 使用说明

这份说明写给第一次使用 AI Canvas 的普通用户。目标是让你知道：怎么启动、怎么配置、怎么创建节点、怎么连线、怎么生成内容、怎么保存和找回文件。

先记住一个核心概念：

> AI Canvas 是一张可以自由摆放节点的创作画布。节点就是一个功能卡片，连线就是把上一个节点的内容交给下一个节点使用。

---

## 1. 快速开始

### 1.1 启动软件

如果你使用的是 Windows 一键整合包：

1. 解压整合包，建议放到简单路径，例如 `D:\AI-CanvasPro`。
2. 第一次启动时，右键 `双击运行.bat`，选择“以管理员身份运行”。
3. 等待启动窗口运行，不要马上关闭这个窗口。
4. 浏览器会自动打开；如果没有自动打开，手动访问：

```text
http://localhost:8777
```

后续启动一般直接双击 `双击运行.bat` 即可。

如果你是源码运行用户，请看项目根目录的 `README.md`，这里不展开开发环境配置。

### 1.2 第一次必须配置 API Key

AI Canvas 本身是画布工具，真正生成文本、图片、视频、音频时，需要调用对应平台的模型，所以第一次使用通常要先配置 API Key。

入口：

1. 点击左侧或左下角的“设置”按钮。
2. 进入“API 输入”。
3. 找到你要使用的平台。
4. 填写 API Key 或接口地址。
5. 点击“保存”。

常见平台：

| 平台 | 常见用途 |
| --- | --- |
| APIMart | 文本、图像等聚合接口 |
| GRSAI | 图像生成等 |
| PPIO 派欧云 | 文本、图像等 |
| RunningHUB | 工作流、图像、视频、音频相关能力 |
| OpenAI 兼容接口 | 兼容 OpenAI 格式的第三方模型 |
| 即梦 | 如果你的版本显示该入口，可通过扫码登录 |

示例只写成这样：

```text
sk-xxxxxxxxxxxxxxxx
Bearer Token...
https://api.example.com/v1/chat/completions
```

不要把自己的真实 API Key 发给别人，也不要公开截图包含 Key 的设置页面。

### 1.3 保存第一个项目

完成操作后按：

```text
Ctrl + S
```

输入项目名称并保存。项目通常会保存到：

```text
user/Canvas Project/
```

注意：项目文件保存的是画布结构、节点、连线和路径信息；生成出来的图片、视频、音频一般在 `output/` 目录。

## 2. 界面认识

### 2.1 顶部区域

顶部主要用于管理项目和画布页。

- 项目名称：可以看到当前项目名，也可以重命名。
- 画布 Tab：一个项目里可以有多个画布页面。
- 新建画布：适合把“人物设定”“场景设定”“视频生成”等内容分开放。

### 2.2 左侧工具栏

左侧是最常用入口。

- 添加：创建文本、图像、视频、音频、3D 场景、360 全景图等节点。
- 项目入口：新建或切换画布项目。
- 资产：管理可复用素材组合。
- 工作流：保存和复用一套节点流程。
- 文件管理：如果你的版本显示该入口，可用于文件相关管理。
- 设置：配置 API、订阅、快捷键和画布偏好。

### 2.3 中间画布

中间就是创作区域。你可以在这里：

- 放置节点。
- 拖动节点。
- 缩放和平移视图。
- 连接节点。
- 查看生成结果。

画布很大，如果节点找不到了，按 `F` 适应画布。

### 2.4 左下角控制

左下角常见控制：

- 小地图：查看节点整体位置。
- 网格吸附：让节点移动时更容易对齐。
- 适应画布：自动把节点拉回视野。
- 缩放滑杆：调整画布显示大小。

## 3. 基础操作

### 3.1 创建节点

常用方式：

1. 点击左侧“添加”。
2. 选择节点类型。
3. 节点会出现在画布上。

也可以把图片、视频、音频文件直接拖进画布，系统会自动创建对应的源节点。

### 3.2 移动画布和节点

- 鼠标滚轮：缩放画布。
- 按住空格拖动：平移画布。
- 按住鼠标中键拖动：平移画布。
- 拖动节点：移动节点。
- `Shift + 点击`：多选节点。
- 框选：在空白处拖出选框，选中多个节点。

### 3.3 连线

连线表示“把一个节点的结果交给另一个节点使用”。

常规步骤：

1. 找到上游节点，例如一张参考图。
2. 从它的连接点拖到下游 AI 节点。
3. 下游节点会收到这个参考内容。

如果节点里有“连接/拾取”按钮，也可以点击按钮后再点目标节点完成连接。

常见用法：

- 源图像 -> AI 图像生成节点。
- 生成图片 -> AI 视频生成节点。
- 源文本 -> AI 文本生成节点。
- 源音频 -> AI 音频生成节点。

### 3.4 导入和导出

导入：

- 把 AI Canvas 的 `.json` 项目文件拖进页面。

导出：

- 在项目或画布菜单里选择“另存为”或导出，保存为 `.json`。

提醒：JSON 不一定包含媒体文件本体。如果要完整迁移项目，最好同时备份 `output/` 和上传素材。

## 4. 节点类型

### 4.1 源节点

源节点用于放已有素材。

| 节点 | 用途 |
| --- | --- |
| 源文本 | 放剧情、提示词、角色设定、文案 |
| 源图像 | 放参考图、人物图、产品图、生成结果 |
| 源视频 | 放视频素材，用于裁剪、抠像、场景检测或参考 |
| 源音频 | 放音频素材，用于裁剪、参考音色等 |

源节点通常是流程的起点。

### 4.2 AI 生成节点

AI 生成节点用于产出新内容。

| 节点 | 用途 |
| --- | --- |
| AI 文本 | 写文案、总结、改写、提取信息 |
| AI 图像 | 文生图、图生图、海报、角色图、场景图 |
| AI 视频 | 文生视频、图生视频、让静态图动起来 |
| AI 音频 | 文本转语音、旁白、音色相关生成 |

基本流程都差不多：

1. 创建 AI 节点。
2. 选择模型。
3. 输入提示词。
4. 需要参考素材时，把源节点连进来。
5. 点击生成。

### 4.3 其他常用节点

| 节点 | 作用 |
| --- | --- |
| 注释节点 | 写说明、备注、待办，方便整理画布 |
| 组节点 | 把多个节点打包管理，适合复杂流程 |
| 分镜节点 | 把多张图排成分镜或宫格图 |
| 场景检测节点 | 分析视频镜头切换，辅助裁剪片段 |
| 3D导演台 | 摆放对象、调机位、截图，适合进阶场景创作 |
| 360 全景图 | 查看或处理全景图，适合全景场景相关流程 |

这些功能不是一开始必须学。新手可以先掌握“源图像 + AI 图像 + AI 视频”这条主线。

## 5. 一个完整的新手流程

下面用“参考图生成海报，再生成视频”举例。

### 5.1 上传参考图

把一张图片拖进画布。系统会创建源图像节点。

### 5.2 创建图像生成节点

1. 点击左侧“添加”。
2. 选择“图像”。
3. 创建 AI 图像生成节点。

### 5.3 连接参考图

把源图像节点连到 AI 图像生成节点。

如果连接成功，AI 图像节点里通常会显示参考图缩略图。

### 5.4 输入提示词

示例：

```text
电影海报风格，雨夜城市，人物站在霓虹灯下，神秘、悬疑、高级质感，画面清晰，光影丰富。
```

提示词可以按这个思路写：

- 主体：画面里有什么。
- 场景：在哪里。
- 风格：写实、电影感、插画、海报等。
- 氛围：温暖、悬疑、梦幻、未来感等。
- 质量：清晰、细节丰富、光影自然。

### 5.5 点击生成

选择合适模型后点击生成。图片生成完成后会显示在节点中，文件通常会落到 `output/`。

### 5.6 继续生成视频

1. 创建 AI 视频节点。
2. 把生成图片连到视频节点。
3. 输入视频提示词。

示例：

```text
镜头缓慢推进，人物轻轻转头看向远方，雨水落下，霓虹灯反光，电影感，动作自然。
```

视频生成通常比图片更慢，也更消耗额度，生成时不要频繁刷新页面。

### 5.7 保存项目

按 `Ctrl + S` 保存项目。建议用清楚的名字，例如：

```text
雨夜侦探海报和视频_v1
```

## 6. 提示词、`@` 引用和 `/` 预设

### 6.1 提示词

提示词就是你告诉 AI 的要求。

文本提示词示例：

```text
请根据上游剧情，提取人物、场景、道具和冲突点，用表格输出。
```

图像提示词示例：

```text
写实电影感，一位年轻侦探站在雨夜街头，背景是蓝红霓虹灯，半身构图，悬疑氛围，清晰。
```

视频提示词示例：

```text
镜头缓慢向前推进，人物抬头看向远处，风吹动衣摆，背景灯光轻微闪烁，动作自然。
```

### 6.2 `@` 引用

在提示词输入框输入 `@`，可以引用其他节点的内容。

适合：

- 引用某个文本节点的设定。
- 引用上一步 AI 文本生成结果。
- 让当前节点基于别的节点继续处理。

例子：

```text
请根据 @角色设定 生成角色半身设定图，写实电影感。
```

### 6.3 `/` 预设

在提示词输入框输入 `/`，可以打开预设菜单。

适合：

- 不知道怎么写提示词时先用模板。
- 快速插入常用指令。
- 保持常用格式统一。

不同节点显示的预设可能不同，以当前界面为准。

### 6.4 连线和 `@` 怎么选

简单记：

- 图片、视频、音频这类素材，优先用连线。
- 文本内容、说明、设定，常用 `@` 引用。
- 不确定时先用连线，因为节点里能直接看到引用缩略图。

## 7. 节点工具栏

选中节点后，节点附近通常会出现工具栏。不同节点的工具不同。

### 7.1 图片工具

常见能力：

- 裁剪：截取图片的一部分。
- 标注：在图片上圈画修改意见。
- 扩图：把画面向外补大。
- 高清：提升清晰度或放大。
- 遮罩编辑：告诉 AI 哪些区域要处理。
- 宫格裁剪：把图片切成多格。
- 全屏显示：放大查看。
- 下载：保存图片。

这些工具是否可用，取决于当前版本、模型和配置。

### 7.2 视频工具

常见能力：

- 裁剪视频：截取一段时间范围。
- 抠像：把人物或目标从背景中分离。
- 高清：修复或提升视频清晰度。
- 全屏显示：放大预览。
- 下载：保存视频。

视频工具通常比图片工具更耗时。处理大视频前，建议先用短片段测试。

### 7.3 音频工具

常见能力：

- 播放和暂停。
- 裁剪音频。
- 调整倍速。
- 下载音频。

### 7.4 文本和注释工具

文本节点常用：

- 复制文本。
- 全屏查看长文本。
- 清理空行，具体以当前版本显示为准。

注释节点常用：

- 编辑内容。
- 改字号。
- 改文字颜色或背景颜色。
- 复制或删除注释。

### 7.5 3D 和全景工具

常见能力：

- 移动、缩放、旋转对象。
- 重置视角。
- 截图。
- 切换机位。

这是进阶功能，新手可以先不用急着学。

## 8. 项目、资产和工作流

### 8.1 项目

项目就是当前画布内容。保存后通常在：

```text
user/Canvas Project/
```

建议项目名写清楚，例如：

```text
短剧_人物设定_v1
产品海报_主视觉_v2
图生视频测试_雨夜场景
```

### 8.2 资产

资产可以理解成“可复用素材包”。

适合保存：

- 一个角色设定。
- 一组场景参考图。
- 一组常用节点。
- 一个常用物品素材。

基本用法：

1. 选中一个或多个节点。
2. 打开资产面板。
3. 创建资产，填写名称和分类。
4. 以后可以从资产面板“添加整个资产”。

### 8.3 工作流

工作流可以理解成“一套可复用流程”。

适合保存：

- 从文本提取人物设定的流程。
- 从参考图生成角色图的流程。
- 从图片继续生成视频的流程。
- 团队常用模板。

基本用法：

1. 在画布上搭好一套节点流程。
2. 打开工作流面板。
3. 创建工作流，填写名称、标签、备注。
4. 需要时再应用到画布。

简单区别：

| 需求 | 推荐 |
| --- | --- |
| 保存素材组合 | 资产 |
| 保存操作流程 | 工作流 |
| 保存人物、场景、物品 | 资产 |
| 保存一套节点生成模板 | 工作流 |

## 9. 设置里常用的东西

### 9.1 API 输入

配置各平台 API Key。生成失败时，优先检查这里。

### 9.2 订阅中心

如果你的版本显示“订阅中心”，可以查看订阅状态、到期时间，或输入授权码。是否需要订阅取决于具体功能和模型。

### 9.3 键盘快捷键

可以修改快捷键，也可以切换预设。新手建议先用默认，熟悉后再改。

### 9.4 画布偏好

常见设置：

- 画布底色。
- 网格点显示。
- 网格吸附。
- 辅助线吸附。
- 多选对齐。
- 图像和视频节点是否允许拖拽缩放。
- 节点创建间距。
- 连续生成方向。

这些设置主要影响使用手感，不影响 AI 模型本身。

## 11. 文件保存在哪里

### 11.1 项目文件

```text
user/Canvas Project/
```

保存画布结构、节点、连线、提示词和路径信息。

### 11.2 上传素材

```text
data/uploads/
```

保存你拖入或上传的图片、视频、音频等素材。

### 11.3 生成结果

```text
output/
```

保存 AI 生成或处理后的图片、视频、音频。

常见子目录：

| 目录 | 可能保存什么 |
| --- | --- |
| `output/CutVideo/` | 裁剪后的视频 |
| `output/CutAudio/` | 裁剪后的音频 |
| `output/SceneCuts/` | 场景检测切出来的视频片段 |
| `output/VideoThumbs/` | 视频缩略图 |

不要随便删除 `output/`。如果删掉，项目 JSON 还在，但节点可能找不到图片或视频。

### 11.4 用户配置

```text
user/config.json
user/settings.json
user/shortcuts.json
```

其中 `user/config.json` 可能保存 API Key，不要公开分享。

### 11.5 建议备份

如果要完整备份项目，建议至少备份：

- `user/Canvas Project/`
- `output/`
- `data/uploads/`
- 如果用了资产，再备份 `data/assets/`

## 12. 常见问题

### 12.1 页面打不开

检查：

- `双击运行.bat` 是否正在运行。
- 浏览器地址是否是 `http://localhost:8777`。
- 启动窗口有没有报错。
- 是否被杀毒软件或防火墙拦截。

可以尝试重新运行 `双击运行.bat`，第一次用管理员身份运行。

### 12.2 生成失败

优先检查：

- 对应平台 API Key 是否填写。
- Key 是否保存。
- 选择的模型是否属于你已配置的平台。
- 平台余额或权限是否足够。
- OpenAI 兼容接口地址是否正确。

### 12.3 配了 Key 还是报错

常见原因：

- 粘贴时多了空格。
- 填错平台。
- Key 已失效。
- 模型需要订阅或会员权限。
- 平台接口临时不可用。

### 12.4 图片或视频不显示

可能是文件被移动或删除了。先检查 `output/` 或 `data/uploads/` 里对应文件是否还在。

如果文件还在，可以尝试刷新页面；如果文件已经没了，只能从备份恢复或重新生成。

### 12.5 拖入 JSON 没反应

可能原因：

- 文件不是 AI Canvas 项目格式。
- JSON 文件损坏。
- 当前页面处于特殊模式。

可以先按 `Esc`，再拖一次。

### 12.6 快捷键没反应

可能原因：

- 当前焦点在输入框。
- 快捷键被你修改过。
- 当前处于裁剪、标注、抠像等特殊模式。

先点一下画布空白处，或按 `Esc` 退出当前模式。

### 12.7 别人打开我的项目看不到图片

通常是因为你只发了 JSON，没有发媒体文件。

分享项目时，最好同时提供相关的 `output/` 文件和上传素材。

### 12.8 路径有中文会不会出问题

很多情况下可以运行，但为了减少问题，普通用户建议放在简单路径，例如：

```text
D:\AI-CanvasPro
```

不要在压缩包里直接运行，也不要放在权限很复杂的目录。

## 13. 推荐学习顺序

新手建议按这个顺序学：

1. 启动软件，打开页面。
2. 配置 API Key。
3. 创建 AI 图像节点，生成一张图。
4. 上传参考图，并连到图像节点。
5. 把生成图连到视频节点，做一次图生视频。
6. 学会 `Ctrl + S` 保存项目，知道文件在 `output/`。
7. 用注释和组节点整理画布。
8. 学资产和工作流，复用常用素材和流程。
9. 再学习裁剪、标注、扩图、高清、抠像、场景检测、3D 场景等进阶工具。

不用一开始就掌握所有功能。先跑通“创建节点 -> 写提示词 -> 连线 -> 生成 -> 保存”这一条主线，就已经能完成大部分基础创作了。
